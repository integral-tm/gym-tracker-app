# Gym Tracker App

This is the frontend part of a mobile application designed for tracking gym
workouts.

## Project Overview

The Gym Tracker app is designed to help users track their workouts, manage their
progress, and achieve their fitness goals.

## Technology Stack

The project is built using a modern technology stack to ensure high performance
and a smooth development experience:

- React Native
- Expo
- Expo Router
- Nativewind
- Clerk

## Getting Started

Follow these instructions to get the project up and running on your local
machine.

#### Prerequisites

Before you begin, ensure you have the following tools installed:

- Node.js (LTS version recommended)
- npm or yarn
- Expo CLI

## Installation

1.  Clone the repository:
   
    ```git clone https://github.com/integral-tm/gym-tracker-app.git```
2. Go to the main directory:
   
    ```cd gym-tracker-app```
3.  Install the project dependencies:
   
    ```npm install``` **or** ```yarn install```

## Running the Application

After installing the dependencies, you can run the application:

1. Start the Expo development server:

    ```npx expo start```

    This command will start the Expo development server and display a QR code directly in your terminal.

2.  Choose how you want to run the app:

    **Android:** Scan the QR code with the Expo Go app on your Android device,
    or press a in the terminal to run it on an Android emulator (if
    installed).

    **iOS:** Scan the QR code with the Expo Go app on your iOS device, or press
    i in the terminal to run it on an iOS simulator (if Xcode is installed).

    **Web:** Press w in the terminal to run the app in your web browser.


Create a **.env** file in the root directory of your project based on .env.example
and add any necessary environment variables.
